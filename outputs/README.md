# Outputs Folder

This folder contains all generated outputs from the Credit Scoring Model pipeline.

## Files

- metrics.txt  
  Contains classification report, confusion matrix, and ROC-AUC score.

- roc_curve_data.pkl  
  Saved ROC curve data (FPR, TPR, thresholds) for plotting and analysis.

## Notes

- Files in this folder are auto-generated by src/main.py
- Do not manually edit output files
- This folder is useful for reporting and evaluation
